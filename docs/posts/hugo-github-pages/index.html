用hugo在github上建立个人博客

  • Mar 29, 2021
  • #hugo , #markdown
  • One minute read

hugo把markdown写的博客文章，转换成html静态网页。将这些静态网页推送到github pages，就可以轻松建立个人博客。现在的这篇文章就是这样生成的。

查看hugo的文档到这里，可以快速了解hugo的使用。

hugo需要选用一个主题，这里使用的是kiera。

快速入门这个主题，可以查看kiera下的exampleSite目录，参考里面的示例来进行配置。
